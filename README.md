## Overview

**SEMOD Lite** is an open-source Escape the Backrooms mod project focused on useful multiplayer and gameplay utilities while keeping the menu lightweight and easy to use.

The mod provides a built-in menu named **SEDET MOD MENU Lite Version**. By default, the menu can be opened with **F3**, and the hotkey can be changed inside the mod menu settings page.

Repository: **https://github.com/SEDET666/SEMOD-Lite**

---

## Main Features

### 16-Player Room Support

SEMOD Lite unlocks the original maximum player limit and allows the room player count to be raised from **4 players** to **16 players**.

![Max player slider set to 16](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/MaxPlayer16.png)

![16-player lobby list](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/MaxPlayer16_2.png)

### Mod Menu Pages

SEMOD Lite includes several menu pages for different functions.

#### Map Page

The Map page contains map loading and room-related options, including the **MaxPlayers** value.

![SEMOD Lite Map page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_1.png)

#### Actor Page

The Actor page provides actor, entity, and item related tools, such as spawning entities/items, destroying spawned entities, and cleaning up items or entities.

![SEMOD Lite Actor page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_2.png)

#### Setting Page

The Setting page contains general mod settings. The default open menu key is **F3**, and the destroy actor key is shown as **Num -** in the screenshot. Options such as **MaxPlayers16**, **CanJoin**, and **Is English** are available here.

![SEMOD Lite Setting page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_3.png)

#### Player Page

The Player page contains player utility functions, including teleport-related tools, free camera, collision toggle, name tag toggle, firework safety toggle, stamina option, and movement speed controls.

![SEMOD Lite Player page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_4.png)

## Default Controls

| Function | Default Key / Location |
| --- | --- |
| Open mod menu | **F3** |
| Change menu key | Mod Menu → **Setting** page |
| Destroy actor key | **Num -** by default, configurable from the Setting page |

---

## Feature List

| Category | Features |
| --- | --- |
| Multiplayer | Unlock maximum player count from 4 to 16, CanJoin option |
| Map | Map selection/loading, MaxPlayers setting, level-related options |
| Actor | Spawn entities, destroy spawned entities, spawn items, use items, clean up entities/items |
| Player | Get location, teleport, teleport all players, free camera, no collision toggle, name tag toggle, stamina option, movement speed controls |
| Settings | Change menu hotkey, destroy actor hotkey, MaxPlayers16 toggle, CanJoin toggle, language toggle |

## Language

SEMOD Lite supports an English interface through the **Is English** option in the Setting page.

## Full Version SEMOD

**SEMOD Lite** is the open-source lightweight version. The full version of **SEMOD** is **not open source**.

You can download the full version here: **https://www.nexusmods.com/escapethebackrooms/mods/111**

## Build Requirement

To compile this project from source, you must use the **ETBAssetGenPlus** project to generate and complete the required Escape the Backrooms game asset files first.

Because the original game assets are not included in this repository, the project may fail to compile or package correctly until the missing game assets have been generated/completed with **ETBAssetGenPlus**.
