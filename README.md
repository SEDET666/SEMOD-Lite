## ✨ Overview

**SEMOD Lite** is the lightweight open-source version of the SEMOD mod menu for **Escape the Backrooms**.

It focuses on practical multiplayer and gameplay features while keeping the menu simple, readable, and easy to use. The mod adds an in-game menu named **SEDET MOD MENU Lite Version**, which can be opened by default with **F3**.

> 🔧 The default menu key is **F3**. You can change it in the mod menu under the **Setting** page.

Repository: **https://github.com/SEDET666/SEMOD-Lite**

---

## 🚀 Highlights

- 👥 Unlocks the maximum lobby player count from **4** to **16**
- 🧭 Includes a built-in map/loading page
- 🧱 Provides actor, entity, and item tools
- 🧍 Includes player utilities such as teleport, free camera, no collision, name tag toggle, and speed settings
- ⚙️ Includes configurable hotkeys and gameplay toggles
- 🌍 Supports language switching through the **Is English** option
- 🧩 Open-source Lite version under the **GPL license**

---

## 👥 16-Player Room Support

The original game lobby is limited to **4 players**. SEMOD Lite adds a **Max Players** control and allows the room limit to be raised to **16 players**.

![Max player slider set to 16](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/MaxPlayer16.png)

After enabling the expanded player limit, the lobby can display more player slots for larger rooms.

![16-player lobby list](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/MaxPlayer16_2.png)

## 🧰 Mod Menu Pages

SEMOD Lite is organized into several pages. Each page focuses on a different type of tool.

### 🗺️ Map Page

The **Map** page contains map loading and room-related options. It includes the **MaxPlayers** value and level-related utilities such as switching Level 94 between day and night and skipping the bunker video.

![SEMOD Lite Map page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_1.png)

### 🧱 Actor Page

The **Actor** page provides actor, entity, and item tools, including:

- Entity spawning
- Spawned entity destruction
- Item spawning
- Item use actions
- Clean up all items
- Clean up all entities
- Spawn quantity control
- Location and scale input fields

![SEMOD Lite Actor page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_2.png)

### ⚙️ Setting Page

The **Setting** page contains general mod configuration options.

From this page, you can configure:

- **Open menu key**: default is **F3**
- **Destroy actor key**: shown as **Num -** in the screenshot
- **MaxPlayers16** toggle
- **CanJoin** toggle
- **Is English** language toggle

![SEMOD Lite Setting page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_3.png)

### 🧍 Player Page

The **Player** page provides player-related utility functions, including:

- Get player location
- Teleport
- Teleport all players
- Free camera
- No collision toggle
- Name tag toggle
- Firework safety toggle
- Stamina option
- Walk speed control
- Sprint speed control
- Apply settings to one player or all players

![SEMOD Lite Player page](https://raw.githubusercontent.com/SEDET666/SEMOD-Lite/refs/heads/main/Pics/SEMOD_Lite_4.png)

## 🛠️ Build Requirement

To compile this project from source, you must use the **ETBAssetGenPlus** project to generate and complete the required **Escape the Backrooms** game asset files first.

Because the original game assets are not included in this repository, the project may fail to compile or package correctly until the missing game assets have been generated/completed with **ETBAssetGenPlus**.
